# 🚀 Abhijith CS | Developer Portfolio

A sleek, professional, and high-performance portfolio website built to showcase modern web development skills. This project is designed with a focus on clean UI, smooth animations, and fast loading speeds.

---

## 🛠 Tech Stack
* **React (Vite)** - For a lightning-fast development experience.
* **Tailwind CSS** - For modern, utility-first styling.
* **Framer Motion** - For production-ready animations and transitions.
* **Lucide React** - For clean and consistent iconography.

---

## ⚡ Quick Start

### 1. Prerequisites
Ensure you have the following installed:
* [Node.js](https://nodejs.org/) (v18 or later)
* npm (comes with Node) or Yarn

### 2. Installation
Clone the repository or extract the project folder, then install the necessary dependencies:
